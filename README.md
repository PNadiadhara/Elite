# Elite
The app Designed to create healthy competitve competition amongst users

## Installation
```bash
pod 'SnapSDK'
pod 'Firebase/Core'
pod 'Firebase/Auth'
pod 'Firebase/Firestore'
pod 'Firebase/Storage'
pod 'Kingfisher'
source 'https://github.com/CocoaPods/Specs.git'
pod 'GoogleMaps'
pod 'GooglePlaces'
```
